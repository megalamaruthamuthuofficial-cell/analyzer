# 🛒 Product Sentiment Analyzer (Flask + React + MongoDB)

A full-stack AI-powered web application that scrapes product reviews from e-commerce websites (Amazon/Flipkart), analyzes sentiment using NLP, and visualizes results using charts.

---

## 🚀 Features

- 🔗 Paste product URL (Amazon/Flipkart)
- 🕷️ Scrapes real user reviews using Selenium
- 🤖 Sentiment analysis using TextBlob
- 📊 Live dashboard with Pie Chart (Positive / Neutral / Negative)
- 🗄️ Stores data in MongoDB
- ⚡ REST API using Flask
- 🎨 Interactive React frontend UI

---

## 🧠 Tech Stack

### Frontend:
- React.js
- Axios
- Chart.js (react-chartjs-2)
- CSS

### Backend:
- Python Flask
- Flask-CORS
- TextBlob (NLP)
- Selenium (Web Scraping)

### Database:
- MongoDB Atlas
